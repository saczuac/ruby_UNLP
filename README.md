## Estándares de codificación **Ruby** 


### Variables de Instancia  

```ruby
@var
```

### Variables de Clase 

```ruby
@@var
```

### Variables Globales 

```ruby
$var
```

### Métodos 

```ruby
def to_s (m = 0)
    m*2
end
```

### Clases 

```ruby
class Person
   @@no_of_dogs=0
   def initialize(id, name)
        @p_id=id
        @p_name=name
    end
    def say_hello(name='Sacha')
        puts "Hello! #{name}"
    end
end 
```
